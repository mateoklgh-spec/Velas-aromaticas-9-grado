# 🕯️ Velas Aromáticas - Proyecto Educativo 9° Grado

## 📋 Descripción del Proyecto

Este es un proyecto educativo desarrollado por estudiantes de 9° grado del **Colegio Parroquial Medalla Milagrosa** ubicado en **Fernando de la Mora, Paraguay**.

El proyecto trata sobre la **elaboración de velas aromáticas de manera artesanal**, considerando elementos como:
- ✨ La forma
- 🎨 El color
- 👃 El aroma

Las velas aromáticas no solo son un bello adorno e iluminación, sino que también:
- �� Son una excelente opción de regalo
- 💼 Representan una oportunidad de negocio
- 🌿 Contribuyen al bienestar de la salud
- 🧘 Generan una atmósfera de paz, relajación y tranquilidad

## 🎯 Objetivo Educativo

El presente proyecto tiene como objetivo describir y dar a conocer los aspectos relacionados con la elaboración de velas aromáticas de manera artesanal, demostrando que la originalidad, la creatividad, el diseño, la experimentación con nuevos materiales y la innovación son aspectos fundamentales para desarrollar productos atractivos y funcionales.
